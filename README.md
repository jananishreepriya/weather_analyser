Features

Dashboard Overview

- Displays current weather for multiple cities.
- Shows temperature, weather condition, and emoji icon.
- Supports Celsius, Fahrenheit toggle.
- Highlights favorite cities (persisted with `localStorage`).

Detailed Forecast View
- Click on any city to open a modal with detailed weather analytics.
- Fetches a 5-day / 3-hour forecast using the OpenWeatherMap API.
- Displays interactive charts with:
  - Temperature
  - Wind Speed
  - Precipitation (Rain)
